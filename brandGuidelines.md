# PawfectNotes Brand Guidelines

## Brand Name
**PawfectNotes**

Always written as one word with a capital "P" and capital "N". The word "Pawfect" is a deliberate playful blend of "Paw" and "Perfect" — never spell it as "Pawfect Notes" (two words) or "PerfectNotes".

### Acceptable Usage
- PawfectNotes
- PAWFECTNOTES (all caps, use sparingly — headlines only)
- pawfectnotes (all lowercase, only in URL/handle contexts)

### Unacceptable Usage
- Pawfect Notes (two words)
- PawfectNote (missing the "s")
- pawfectNotes (lowercase "p")
- Pawfect_Notes (underscore)
- PawFectNotes (wrong capitalization)

---

## Brand Identity

### Concept
PawfectNotes is a cat-themed digital greeting card and note-sharing platform. The aesthetic draws from the warm, handcrafted, botanical romanticism of digibouquet.vercel.app — but replaces florals with cats, paws, whiskers, and cozy feline motifs. The result is a brand that feels like a handwritten letter tucked inside a cat café: intimate, soft, slightly whimsical, and emotionally resonant.

### Brand Personality
- **Warm** — Like curling up with a cat on a rainy afternoon
- **Handcrafted** — Ink-drawn illustrations, typewriter-style text, tactile paper feel
- **Playful but not childish** — Cat humor with emotional depth
- **Romantic** — Inspired by letters, bouquets, and vintage stationery
- **Minimal** — Clean cream backgrounds; nothing loud, nothing corporate

### Tagline Options
- *"Send love with paws."*
- *"Notes that purr."*
- *"A little note. A lot of love."*

## Logo Usage

### Logo Description
The PawfectNotes logo uses a flowing script wordmark (similar in character to Digibouquet's calligraphic style) paired with a small cat illustration or paw print icon. The cat icon can be a sitting cat silhouette, a paw print in watercolor wash, or a cat face with ink-drawn style consistent with the site's illustration language.

### Clear Space
Maintain clear space around the logo equal to the height of the "P" in the wordmark on all sides.

### Minimum Size
- Digital: 140px width minimum
- Print: 1.2 inches / 30mm width minimum

### Logo Variations
- **Full logo**: Wordmark + cat icon (default)
- **Icon only**: Paw print or sitting cat silhouette (for favicons, app icons)
- **Wordmark only**: For horizontal banners where icon would crowd the layout
- **Black & white version**: For monochrome print contexts

### Logo Don'ts
- Do not stretch or distort the logo
- Do not rotate the logo
- Do not swap the script font for a sans-serif
- Do not recolor the cat illustration outside approved palette
- Do not add drop shadows, glows, or gradients to the wordmark
- Do not place the logo on dark, busy, or photographic backgrounds

## Typography

### Primary Display Font
**Playfair Display** (Google Fonts)
- Use for: Page titles, greeting card headers, brand moments
- Style: Serif with elegant swashes; evokes letterpress printing
- Fallbacks: Georgia, "Times New Roman", serif

### Secondary / Accent Font
**Dancing Script** (Google Fonts)
- Use for: Handwritten-style labels, recipient names, personal touches, decorative subtext
- Style: Flowing cursive; mirrors the calligraphic energy of the Digibouquet wordmark
- Fallbacks: Brush Script MT, cursive

### Body Font
**Lora** (Google Fonts)
- Use for: Body copy, card message text, descriptions, UI labels
- Style: Contemporary serif; warm and literary
- Fallbacks: Palatino, "Book Antiqua", serif

### Monospace / Typewriter Font
**Courier Prime** (Google Fonts)
- Use for: Card message preview (typewriter-style rendering), transaction IDs, code-like UI elements
- Style: Classic typewriter; evokes handcrafted letters
- Fallbacks: "Courier New", monospace

---

### Font Weights
- **Bold (700)**: Display headings, page titles, CTA buttons
- **Semi-Bold (600)**: Section headers, card titles, form labels
- **Regular (400)**: Body text, descriptions, UI copy
- **Italic (400i)**: Emphasis within card messages, pull quotes

### Font Sizes
- Display / Hero: 48px / 3rem
- H1: 36px / 2.25rem
- H2: 28px / 1.75rem
- H3: 22px / 1.375rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Caption / Label: 12px / 0.75rem

### Line Height
- Display / Headings: 1.15
- Body: 1.65
- Card message text: 1.8 (generous spacing for emotional readability)

### Letter Spacing
- Display (Playfair): 0px (use natural kerning)
- All-caps UI labels: 0.15em (matches Digibouquet's tracked uppercase style)
- Body (Lora): 0px

## Illustration Style

### Core Style
All illustrations follow an **ink-line + watercolor wash** style — hand-drawn outlines in dark ink with loose, slightly-outside-the-lines watercolor fills. This matches Digibouquet's floral illustration aesthetic but replaces flowers with cats.

### Subject Matter
- Sitting cats (various poses: curled, stretching, looking up)
- Paw prints (single, scattered, trail)
- Cat faces / expressions (sleepy, happy, surprised)
- Yarn balls and string tangles
- Cat tails and whisker details
- Small fish or mouse charms (as decorative accents)
- Cat ear silhouettes
- Tiny heart paw prints

### Illustration Do's
- Use thick black outlines with slight imperfection (hand-drawn feel)
- Apply soft watercolor washes inside the outlines — blush, lavender, sage, golden
- Leave some white space inside washes for a "light catching" effect
- Vary sizes across UI (small for stickers, large for hero moments)

### Illustration Don'ts
- Do not use flat vector/cartoon-style fills — no solid fills without line texture
- Do not use digital-looking gradients on illustrations
- Do not use pixel-perfect symmetry — slight imperfection is intentional

## Spacing & Layout

### Base Unit
**8px** — all spacing is a multiple of 8

### Layout Principles
- **Centered, single-column layouts** for emotional/greeting content — mirrors Digibouquet's focused flow
- **Generous white (cream) space** — let the illustrations breathe
- **Flanking illustration placement** — place cat illustrations on left and right of card content (as seen in Digibouquet's card-writing step)
- Avoid dense multi-column grids for primary content; use them only for flower/sticker picker grids

### Step Wizard Layout
- One task per screen — no overwhelming multi-panel forms
- Progress indicator at top (step count, minimal dots or text)
- BACK / NEXT buttons always at bottom center, flat-style

---

## Motion & Animation

### Philosophy
Motion should feel like something being gently placed, not snapping or zooming. Soft, organic, unhurried.

### Transitions
- Page transitions: fade-in, 300ms ease
- Button hover: color shift only, 150ms
- Card entrance: fade up (translateY -10px → 0), 400ms ease-out

### Special Animations
- **Paw print trail**: On greeting page load, small paw prints fade in sequentially across the bottom of the screen (like a cat walked through)
- **Cat ear wiggle**: Logo cat icon gently rotates ears on hover (CSS keyframe, subtle)
- **Bloom picker selection**: Selected cat sticker gets a soft circular highlight pulse (same energy as Digibouquet's selected flower badge)

### Animation Don'ts
- No bouncing or spring physics
- No fast snap animations
- No parallax scrolling effects
- No looping animations that run without user interaction

## Iconography

### Style
- Outlined, hand-drawn feel — match illustration line weight
- Avoid geometric perfection; slight imperfection preferred
- Stroke width: 1.5px at 24px size

### Sizes
- Standard: 24px
- Small (inline): 16px
- Large (hero accent): 40px

### Icon Set Theme
Use cat-themed custom icons where possible:
- ✉️ → Replace with paw-print envelope
- ❤️ → Replace with paw-heart (heart with toe beans)
- ⭐ → Cat star shape (ear-tipped)
- 🔗 → Standard link icon (no replacement needed)

## Tone of Voice

- **Warm and tender**: Speak like a friend handing you a card, not a brand pitching a product
- **Playful without being childish**: Cat puns are welcome; avoid baby talk
- **Concise and poetic**: Short sentences. Room to breathe. Emotion over information
- **Second person**: Always address the user as "you" — intimate and direct

### Vocabulary Examples
| Generic | PawfectNotes |
|---------|--------------|
| "Create a card" | "Write a note" |
| "Add decorations" | "Pick your blooms" → "Pick your cats" |
| "Share your link" | "Send it with paws" |
| "Loading..." | "Finding the purr-fect layout..." |
| "Error" | "Something went sideways (like a cat off a table)" |
